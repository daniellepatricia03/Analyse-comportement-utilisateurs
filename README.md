#  Analyse du comportement des utilisateurs e-commerce

## Description
Analyse exploratoire d'un dataset réel de 42 millions d'actions 
utilisateurs sur une plateforme e-commerce (octobre 2019).

## Objectif
Comprendre les habitudes d'achat des utilisateurs et extraire 
des insights business actionnables.

## Outils utilisés
- Python
- Pandas
- Matplotlib
- Kaggle Notebooks

## Dataset
- Source : Kaggle — eCommerce behavior data (mkechinov)
- 42 millions de lignes, 9 colonnes
- Données : vues, ajouts panier, achats

## Résultats clés
- **Taux de conversion : 2%** — sur 100 visiteurs, seulement 2 achètent
- **Samsung et Apple** dominent les achats (55% du top 10)
- **Pic d'activité à 16h** — meilleur moment pour les promotions
- **Electronics.smartphone** : catégorie la plus consultée (10M de vues)
